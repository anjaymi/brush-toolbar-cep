## 🇬🇧 Meow Brush Favorites

Efficient Brush Management & Tool Switcher for Adobe Photoshop 2023+ (v24.2+). Organize your tools and switch between them instantly.

![Preview](public/icons/plugin-icon.png)

### ✨ Features
*   **Brush Grid**:
    *   **Visual Favorites**: Grid view of your most used brushes.
    *   **Drag & Drop**: Easily reorder brushes with drag-and-drop.
    *   **Quick Delete**: Hover and click `[x]` to remove.
*   **Tool Switcher**:
    *   **One-Click Access**: Switch between Brush, Eraser, Smudge, and Mixer Brush instantly.
    *   **Properties Panel**: Real-time control of Size, Flow, and Opacity.
*   **Customization**:
    *   **Compact Mode**: Double-click title bar to minimize UI.
    *   **Icon Selector**: Choose custom icons/colors for better organization.

### 🚀 Installation
1.  Download the latest `.ccx` file from **[Releases](https://github.com/anjaymi/brush-toolbar-cep/releases)**.
2.  Double-click the file to install.
3.  Restart Photoshop.

## 🇨🇳 喵笔刷收藏 (Meow Brush Favorites)

高效的笔刷管理与快速调用插件，帮助画师在复杂绘画中随心所欲切换工具，不再为寻找笔刷打断心流。

### ✨ 核心功能
*   **笔刷网格**:
    *   **可视化收藏**: 将常用笔刷以网格形式展示，一目了然。
    *   **拖拽排序**: 支持拖拽调整笔刷顺序。
    *   **右键互换**: 支持右键快速交换位置。
*   **工具切换**:
    *   **一键调用**: 顶部常驻画笔、橡皮、涂抹、混合画笔切换按钮。
    *   **参数面板**: 实时调整当前工具的大小、流量和不透明度。
*   **个性化**:
    *   **因地制宜**: 支持双击标题栏折叠为迷你模式，节省屏幕空间。
    *   **自定义图标**: 为不同笔刷设置不同的图标和颜色，便于记忆。

### 🚀 安装指南
1.  在 **[Releases](https://github.com/anjaymi/brush-toolbar-cep/releases)** 下载最新 `.ccx` 安装包。
2.  双击运行自动安装。
3.  重启 Photoshop 即可使用。

## 🇯🇵 Meow Brush Favorites (ニャー・ブラシお気に入り)

Photoshop 2023+ (v24.2+) 用の効率的なブラシ管理＆ツール切り替えプラグインです。ツールを整理し、瞬時に切り替えることで、制作フローをスムーズにします。

### ✨ 主な機能
*   **ブラシグリッド**:
    *   **ビジュアルお気に入り**: よく使うブラシをグリッド表示で管理。
    *   **ドラッグ＆ドロップ**: ブラシの順序を簡単に並べ替えられます。
    *   **クイック削除**: 簡単にリストから削除できます。
*   **ツールスイッチャー**:
    *   **ワンクリックアクセス**: ブラシ、消しゴム、指先ツール、混合ブラシを瞬時に切り替え。
    *   **プロパティパネル**: サイズ、流量、不透明度をリアルタイムで制御。
*   **カスタマイズ**:
    *   **コンパクトモード**: タイトルバーをダブルクリックしてUIを最小化。
    *   **アイコン選択**: カスタムアイコンと色を設定して、視認性を向上。

### 🚀 インストール方法
1.  **[Releases](https://github.com/anjaymi/brush-toolbar-cep/releases)** から最新の `.ccx` ファイルをダウンロードします。
2.  ファイルをダブルクリックしてインストールします。
3.  Photoshopを再起動してください。
